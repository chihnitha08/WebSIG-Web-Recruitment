# level-1: BASICS OF GIT AND GITHUB

## what is git?
- git is a distributed version control system. It keeps track of the changes done, and we can go back whenever we need.
- git helps us manage different versions of our project. 
- git saves the changes we made, and if no changes are made in the next version for a file then it stores link to the previous itractdential version.

## what is github?
- Github is a host for git repositories, in simple terms it a web based platform that stores and shares git repositories.
- Github pushes your local git repository to internet.
- Github helps in collaborations and to share your projects publicly and can help build a portfolio.

## why developers us it?
- to track and manage changes done, to collaborate, to store and backup projects, to easily revert mistakes.
